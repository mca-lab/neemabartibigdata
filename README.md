# STATSR – R Lab Exercises

This folder contains R programming exercises completed as part of the R Laboratory coursework.  
The exercises focus on data frames, descriptive statistics, visualization, and basic exploratory data analysis using R.

---

## 📘 Contents

### 1. **EXERCISE.R**
This script includes:
- Creation of a student data frame  
- Calculation of average marks and maximum age  
- Adding a new column “Grade” based on marks criteria  
- Displaying the updated data frame  

### 2. **EXERCISE2.R**
This script uses the built-in `iris` dataset and includes:
- Finding the number of rows and columns  
- Displaying the first 6 rows  
- Calculating mean, median, variance, and standard deviation of *Sepal.Length*  
- Counting the number of flowers per species  
- Plotting:
  - Histogram of *Petal.Length*
  - Boxplot of *Sepal.Width*
